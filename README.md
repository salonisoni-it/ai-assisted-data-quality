AI-Assisted Data Quality Framework

SQL | Python | Power BI | Governance | (Cloud-ready)

**Project Overview**
This project demonstrates an end-to-end data quality framework built using SQL and Python, with AI-assisted fuzzy matching and executive-level reporting in Power BI.

**The framework detects:**
Missing critical fields
Exact and fuzzy duplicates
Invalid reference data
Orphan transactional records
Data quality metrics with governance thresholds

**Tech Stack**
SQL (MySQL) – data profiling & rule validation
Python (Pandas, RapidFuzz) – AI-assisted fuzzy matching & metrics
Power BI – KPI dashboard & data quality monitoring
GitHub – version control & portfolio presentation

**Project Structure**
ai-assisted-data-quality/
Project folder structure:

- sql         → SQL-based data quality checks
- python      → AI-assisted validation and metrics generation
- powerbi     → Dashboard files and CSV metrics
- data        → Sample datasets used in the project
- governance  → Data quality rules and thresholds

**Key Data Quality Metrics**
Total customers
% Missing names
% Missing emails
Exact duplicate emails
Potential duplicate customers (AI fuzzy match)

**Power BI Dashboard**
The dashboard highlights:
Executive KPIs (cards)
Missing data percentages
Data quality risk indicators
Governance threshold breaches
Metrics above 5% are flagged for remediation.

**Future Enhancements**
Cloud ingestion (BigQuery / Azure)
Automated data quality alerts
CI/CD validation checks
Incremental data profiling
